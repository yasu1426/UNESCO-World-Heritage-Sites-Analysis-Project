# UNESCO World Heritage Sites Analysis

## Project Overview

The UNESCO World Heritage Sites Analysis project is an interactive Tableau dashboard developed to analyze UNESCO World Heritage Sites across different countries and regions. The dashboard helps users explore heritage site distribution, categories, danger status, and regional insights through interactive visualizations.

---

## Objectives

- Analyze UNESCO World Heritage Sites.
- Compare heritage sites across countries and regions.
- Identify endangered heritage sites.
- Visualize historical and geographical patterns.
- Provide an interactive dashboard for data exploration.
